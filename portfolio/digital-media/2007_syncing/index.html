Syncing Floor

2007, Interactive Installation
Private Work
Exhibited the interactive installation which connects 2 distant exhibition sites.

info: Exhibited the interactive installation which connects 2 distant exhibition sites. concept: Visitors can feel the bustle of the exhibition site inconspicuously. challenge: I decided to join a media art exhibition in the Univ. of Tokyo. This exhibition has 2 distant sites so I utilized this characteristic and I planned to visualize an atmosphere of the sites. approach: According to the number of visitors, towns in the display will grow. Growing speed is different according to the flow of visitors and buildings or characters in the display are different according to the time of visitors’ coming. Syncing Floor is a installation art which visualizes ‘‘bustle’’ of distant places. result: This installation art was exhibited at iii Exhibition 7, and connected 2 distant exhibition sites by visualising ‘‘bustle’’ of 2 distant exhibition sites(2nd floor and 9th floor). workcat: Private Work pjtype: Interactive Installation phase: Concept, Design, Build, Run period: 05/2007 – 07/2007 role: Media Artist method: Prototyping device: tech: OpenGL, Network Programming, IR Sensor team: Designer (Me) output: Run in Exhibition release: 07/2007

exhibition: yes exhibition1: iii exhibition 7, The University of Tokyo, Japan, 2007.

Concept: Visitors can feel the bustle of the exhibition site inconspicuously.

Design Process: I built all functions from back-end to front-end. Using infrared sensor, I extracted moments that person passed away my artwork. And then I visualized these bustles of 2 distant locations using network programming and OpenGL.

Exhibited in iii exhibition 7, Media art exhibition of The University of Tokyo, Japan, 2007.
